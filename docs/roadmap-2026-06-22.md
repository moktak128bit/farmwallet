# FarmWallet 로드맵 — "기록기 → 어드바이저" (2026-06-22)

> 목표(사용자 결정): **나/우리만의 완벽한 투자·재무 도구**. 서버 없음(localStorage + Gist) 유지, 멀티유저·온보딩 불필요. 투자 핵심에 자원 집중, 운동·데이트 정산 등 개인 기능은 유지하되 신규 투자 없음.

## 진단 한 줄

이미 **훌륭한 "기록기(tracker)"** 는 완성. "내가 뭘 가졌고 얼마 벌었나"는 거의 다 답한다(FIFO 손익·IRR·목표 포트폴리오·리밸런싱 제안·배당 추적·자산 스냅샷). 비어 있는 건 어드바이저가 답하는 질문 — **"시장보다 잘하나? 위험은? 세금에서 얼마 새나? 앞으로 배당은?"**

## 키스톤 — 먼저 깔아야 다 풀린다

**A0. 일별 포트폴리오 평가액 시계열**
벤치마크 곡선·진짜 TWR·MDD·변동성·샤프가 전부 이것 하나에 의존한다.
- 이미 있는 것: `historicalDailyCloses`가 보유종목 **일별 종가**(최근 120일 일별, 이전은 월말) 적립. 거래는 전부 기록. 보유수량은 `computePositions`로 재계산 가능.
- 빠진 것: **일별 환율 이력**(현재 월 2회 marketEnvSnapshots에만). 이것만 더하면 `Σ(보유수량 × 그날 종가 × 그날 환율)`로 일별 평가액 곡선 복원.
- effort: **M**. 신규 타입 `historicalDailyFx?: Array<{date, rate}>` + 시세 갱신 시 환율도 일별 적립 + `buildDailyPortfolioValue(date→value)` 셀렉터.
- ⚠️ AppData 필드 추가 시 dataService.loadData()·tableDataBackup 동시 반영(CLAUDE.md 규칙).

### G1. 미래지향 (키스톤 A0와 직결)
- **순자산/FIRE 시뮬레이터** — "매달 X 투자 시 N년 후 자산", 은퇴 가능 시점, 과거 수익률 기반 몬테카를로 신뢰구간. assetSnapshots·A0 시계열 재활용.
- **목표 기반 플래닝** — 집·은퇴·교육 목표별 필요 저축률 역산, 진행률. investmentGoals 확장.
- **통합 현금흐름 예측** — 정기수입+지출+대출상환+구독+예상배당(C1) → 향후 6~12개월 잔고 곡선 + 마이너스 경고. 기존 cashFlowForecast 통합.

### G2. 한국 특화 심화
- **연말정산 도우미** — 신용카드 소득공제 한도 추적, 의료비/기부금 집계, 카드 vs 현금 최적 사용 가이드. 한국 가계부 킬러.
- **환율 밴드 보조** — USD 매수 시 "최근 1년 대비 비쌈/쌈" 분위수 표시. A0 일별 환율 재활용.

### G3. 지능·자동화
- **넛지/알림 엔진(PWA 푸시)** — 종합과세 임계 근접(B1), 배당락 임박(C1), 예산 페이스, 환율 급변, 종목 급등락.
- **자동 인사이트(anomaly)** — "식비 평소대비 +40%", "안 쓰는 구독 3개" 자동 발견. 기존 insights anomaly 확장.
- **(선택) AI 비서** — 자연어 질의("작년 대비 식비 어때?")·자동 분류 제안·리포트 내러티브. 클라이언트에서 Claude API 직접 호출(서버 없음 유지, 키는 사용자 보관).

### G4. 신뢰·프라이버시 (개인 도구라 중요)
- **앱 잠금** — PIN/생체, 민감금액 블러, 스크린샷 세이프 모드.
- **거래내역 임포트** — 증권사/은행 CSV 반자동 입력. 수기 입력 부담↓.
